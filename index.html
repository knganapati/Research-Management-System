The admin module enables the addition and removal of researchers and projects, ensuring an up-to-date database for efficient management of research endeavors.

The language utilized for the implementation of project should be independent of platform, powerful and simple, object oriented, robust, powerful, interactive, simple to understand and learn, dynamic and extensible. The whole Project is divided into two parts the front end and the back end. The front end is designed using PHP, CSS, HTML. The back end is designed using myself which is used to design the databases.

XAMPP, which stands for Cross-Platform, Apache, MySQL, PHP, and Perl, is a free platform that allows developers to test their code locally on their own computers. This platform provides the experience of having your own mini web server at home, compatible with both Windows (WAMP) and Linux (LAMP) environments.

With user-friendly interface and comprehensive features, the research management system streamlines administrative tasks, enhances collaboration among researchers, and fosters innovation and knowledge dissemination within the academic and scientific community, ultimately advancing the institution's research goals and reputation.

Overview of the Project
